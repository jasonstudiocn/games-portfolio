# JasonStudio Vibe Coding

这是JasonStudio的Vibe Coding项目，展示三个网页游戏的作品集页面，包含游戏介绍、操作说明和快速访问功能。

## 包含游戏

1. **暗区突围** - 2D搜打撤射击游戏
2. **猫咪快跑** - 横版跑酷游戏  
3. **网页版我的世界** - 3D沙盒建造游戏

## 功能特性

- 🎨 现代化的响应式设计
- 📱 移动端友好
- ⌨️ 键盘快捷键支持 (Ctrl+1/2/3 打开游戏，Ctrl+Shift+1/2/3 打开GitHub)
- 👆 触摸滑动支持
- 🔄 平滑滚动动画
- 🔗 直接链接到游戏页面
- 🔗 GitHub仓库快速访问
- 🎯 一键查看项目源码

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 点击游戏卡片或"开始游戏"按钮打开对应游戏（本地文件）
3. 点击"GitHub"按钮查看项目源码
4. 使用键盘快捷键：
   - Ctrl+1/2/3 快速访问游戏（本地文件）
   - Ctrl+Shift+1/2/3 快速访问GitHub仓库

## 📁 游戏文件

- 游戏运行文件已复制到 `games/` 目录
- 确保作品集可独立运行，无需外部依赖
- 使用 `./sync-games.sh` 脚本可同步更新游戏文件

## 🖼️ 游戏图片

- 真实游戏截图已复制到 `images/` 目录
- 替换了原来的CSS渐变背景
- 使用 `./sync-images.sh` 脚本可同步更新游戏图片
- 支持图片点击打开游戏功能
